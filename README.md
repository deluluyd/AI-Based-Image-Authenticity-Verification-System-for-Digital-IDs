# AI-Based Image Authenticity Verification System

This project is a web-based system designed to verify the authenticity of digital ID images using AI-based image analysis. Users can upload ID images for verification, receive authenticity results, and view verification history. The system supports role-based access for administrators, verifiers, and standard users, ensuring secure data handling and reliable identity verification.

## Features

* User registration and authentication
* Digital ID image upload (JPG, PNG)
* AI-based authenticity analysis
* Verification result generation
* Verification history tracking
* Administrative review and account management

## Purpose

To help organizations detect manipulated or AI-generated digital IDs and improve trust in digital identity verification.
